# CRUD Hexagonal PHP + MySQL

Proyecto desarrollado utilizando arquitectura hexagonal, principios de DDD y CQRS.

##  Descripción

Este sistema permite la gestión de usuarios y vehículos, incluyendo autenticación, recuperación de contraseña y operaciones CRUDL, utilizando PHP y MySQL.

---

##  Tecnologías utilizadas

- PHP
- MySQL
- Arquitectura Hexagonal
- DDD (Domain Driven Design)
- CQRS
- Git y GitHub

---

##  Funcionalidades

- ✔ CRUD de usuarios  
- ✔ Inicio de sesión (login)  
- ✔ Recuperación de contraseña  
- ✔ CRUDL de vehículos  

---

##  Módulo Usuarios
Se implemento CRUD de usuarios siguiendo la guia y arquitectura hexagonal.

##  Módulo Login
Se implemento inicio de sesion con validacion de credenciales y sesiones PHP.

##  Recuperación de contraseña
Se implemento recuperacion de contrasena mediante correo y contrasena temporal.

##  Módulo Vehículos
Se implemento CRUDL de vehiculos con persistencia en MySQL.
